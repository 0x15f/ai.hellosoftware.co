# Getting Started

HelloAI is an simple yet powerful cloud based AI platform that allows you to create an ai model powered by multiple supported algos and train them without managing the software or infastructure behind your model.

HelloAI has one primary base endpoint for all authentication and api endpoints. https://ai.hellosoftware.co/api/. All endpoints return a JSON object.
