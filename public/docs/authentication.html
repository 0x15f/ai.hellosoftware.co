# Authentication

HelloAI uses JWT (JSON Web Token) based authentication. You can login and fetch your account details via the API. We don't store any logs for authentication endpoints so your account credentials are protected.

# Login

Authenticate with your email/password.

Endpoint: auth/login

Method: POST

Required Parameters:

Name Type Optional? Description
email string required Your HelloAI account email
password string required Your HelloAI account password

Example Request:

use GuzzleHttp\Client;

$client = new Client();

$auth_request = $client->post('https://ai.hellosoftware.co/api/auth/login', [
    'form_params' => [
        'email' => 'mail@example.com',
        'password' => 'MyV3ryS3cur3P@ssword!',
    ]
]);

Example Response:

{
    "access_token":"eyJ0eXzI1NiJ9.eyJpc1wvAzTQ2YWEifQ.iEs9lWoLIZ9CxlFfpEOqAf4",
    "token_type":"bearer",
    "expires_in":3600
}
